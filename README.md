# server-side-javascript
Templatebase javascript on server side using mozjs and rust lang.

### Example 
### Code Html & Javascript
```html
<html>
  <head></head>
  <body>
    <?js
      //this is javascript embed tag.
      write('hello world');
    ?>
  </body>
</html>
```
